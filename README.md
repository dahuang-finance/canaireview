# canaireview

Living research project for **"Can AI Replace the Peer Reviewer?"** —
Huang, Kalay, and Schallheim (2026).

This repo hosts the public-facing site at
[dahuang-finance.github.io/canaireview](https://dahuang-finance.github.io/canaireview/)
(eventually moving to `canaireview.com`).

The paper itself, code, and data live in a separate research workspace.
This repo holds only the website source.

## Local development

```bash
# Serve locally to preview before pushing
cd website
python3 -m http.server 8000
# then open http://localhost:8000 in a browser
```
